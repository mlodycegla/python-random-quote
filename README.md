# Random python quote

Returns a random quote and it's author in JSON format.

- **URL:** /quote
- **Method:** `GET`

============

- **URL:** /addquote
- **Method:** `POST`
- **Data Params:** `{ 'author': 'the author of the quote', 'quote': 'the quote' }`
